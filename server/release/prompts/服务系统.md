# 服务系统  

ds4viz提供在线Web服务. 在线Web服务使用Python+FastAPI+PostgreSQL作为技术栈.  
后台管理系统在服务器的本地使用TUI, 直接修改数据库, 和后端无关.  
系统部署在Fedora 43.  
前端使用Vue SPA, 也由服务器返回.  

## 用户系统  

用户需要注册并登陆才能使用ds4viz.  
用户可以填写头像和个人资料.  
用户可以被封禁等.  
在默认的Standard级别受约束:  

## 代码运行  

使用systemd-run瞬态单元, 在临时沙箱中安全的运行代码.  
对于原有没有的.toml, 那新多的toml就是生成的toml--名称不一定.  
对于每次执行的历史结果, PostgreSQL 作为缓存系统. API需要返回是否缓存命中.  

代码有超时时间, 也可以在服务器中进行定义.  

语言暂时考虑支持以下, 但是要设计的高可扩展:  

- python(默认)  
- lua  
- rust  

## 模板系统  

系统提供一个模板系统, 包含指定示例.  
模板需要提供多个语言支持--包含系统支持的所有语言.  
每个模板包括示例代码和说明, 包含标题和正文, 使用markdown.  
用户可以收藏模板.
