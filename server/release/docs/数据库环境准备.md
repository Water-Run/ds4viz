# 数据库环境准备文档

## 环境概述

本项目使用 PostgreSQL 18.1 作为数据库，配置了生产环境和测试环境两套独立的数据库。

## 数据库信息

| 环境 | 数据库名 | 用户名 | 密码 | 主机 | 端口 |
|------|----------|--------|------|------|------|
| 生产 | ds4viz_prod | ds4viz | PWD4ds4viz | localhost | 5432 |
| 测试 | ds4viz_test | ds4viz_test | test_pwd_123 | localhost | 5432 |

## 准备步骤

### 1. 启动 PostgreSQL 服务

```bash
sudo systemctl start postgresql
sudo systemctl enable postgresql
```

验证服务状态：

```bash
systemctl status postgresql
```

### 2. 创建用户和数据库

进入 PostgreSQL 交互终端：

```bash
sudo -u postgres psql
```

执行以下 SQL 命令：

```sql
-- 生产环境
CREATE USER ds4viz WITH PASSWORD 'PWD4ds4viz';
CREATE DATABASE ds4viz_prod OWNER ds4viz;

-- 测试环境
CREATE USER ds4viz_test WITH PASSWORD 'test_pwd_123';
CREATE DATABASE ds4viz_test OWNER ds4viz_test;
```

退出终端：

```sql
\q
```

### 3. 配置客户端认证

查看配置文件位置：

```bash
sudo -u postgres psql -c "SHOW hba_file;"
```

编辑 `/var/lib/pgsql/data/pg_hba.conf`：

```bash
sudo nano /var/lib/pgsql/data/pg_hba.conf
```

确保 IPv4 和 IPv6 本地连接使用 `md5` 认证：

```
# IPv4 local connections:
host    all             all             127.0.0.1/32            md5
# IPv6 local connections:
host    all             all             ::1/128                 md5
```

重新加载配置：

```bash
sudo systemctl reload postgresql
```

### 4. 初始化表结构

进入项目 release 目录：

```bash
cd ~/Coding/ds4viz/server/release
```

初始化生产数据库（密码：PWD4ds4viz）：

```bash
psql -h localhost -U ds4viz -d ds4viz_prod -f sql/init.sql
```

初始化测试数据库（密码：test_pwd_123）：

```bash
psql -h localhost -U ds4viz_test -d ds4viz_test -f sql/init.sql
```

## 连接测试

```bash
psql -h localhost -U ds4viz -d ds4viz_prod
psql -h localhost -U ds4viz_test -d ds4viz_test
```

## 常用维护命令

```bash
# 查看服务状态
systemctl status postgresql

# 重启服务
sudo systemctl restart postgresql

# 查看数据库列表
sudo -u postgres psql -c "\l"

# 查看用户列表
sudo -u postgres psql -c "\du"
```
