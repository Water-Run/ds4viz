# 表定义

## 表结构描述

### users 用户主表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 主键 |
| username | VARCHAR(64) | UNIQUE, NOT NULL | 用户名 |
| email | VARCHAR(255) | UNIQUE, NOT NULL | 邮箱 |
| password_hash | VARCHAR(255) | NOT NULL | 密码哈希 |
| avatar | BYTEA | | 头像二进制数据 |
| avatar_mime_type | VARCHAR(32) | | 头像MIME类型 |
| bio | TEXT | | 个人简介 |
| level | VARCHAR(16) | NOT NULL, DEFAULT 'standard' | 枚举：standard/enhanced |
| status | VARCHAR(16) | NOT NULL, DEFAULT 'active' | 枚举：active/banned/suspended |
| banned_reason | TEXT | | 封禁原因 |
| banned_until | TIMESTAMPTZ | | 封禁截止时间，NULL表示永久 |
| created_at | TIMESTAMPTZ | NOT NULL, DEFAULT NOW() | 注册时间 |
| updated_at | TIMESTAMPTZ | NOT NULL, DEFAULT NOW() | 更新时间 |

### level_upgrade_requests 升级申请

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 主键 |
| user_id | UUID | FK → users, NOT NULL | 申请用户 |
| status | VARCHAR(16) | NOT NULL, DEFAULT 'pending' | 枚举：pending/approved/rejected |
| reason | TEXT | | 用户申请理由 |
| reviewer_note | TEXT | | 管理员备注 |
| created_at | TIMESTAMPTZ | NOT NULL, DEFAULT NOW() | 申请时间 |
| reviewed_at | TIMESTAMPTZ | | 审核时间 |

### user_quota_usage 配额追踪

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 主键 |
| user_id | UUID | FK → users, NOT NULL | 用户 |
| period_start | TIMESTAMPTZ | NOT NULL | 周期开始时间 |
| cpu_time_used_ms | BIGINT | NOT NULL, DEFAULT 0 | 已用CPU时间（毫秒） |

联合唯一约束：(user_id, period_start)

### languages 语言配置

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | VARCHAR(32) | PK | 语言标识：python/lua/rust |
| display_name | VARCHAR(64) | NOT NULL | 显示名称 |
| version | VARCHAR(32) | | 当前版本号 |
| is_interpreted | BOOLEAN | NOT NULL, DEFAULT true | 是否解释型语言 |
| file_extension | VARCHAR(16) | NOT NULL | 文件扩展名 |
| enabled | BOOLEAN | NOT NULL, DEFAULT true | 是否启用 |
| timeout_ms | INT | NOT NULL, DEFAULT 30000 | 执行超时时间（毫秒） |
| sort_order | INT | NOT NULL, DEFAULT 0 | 显示排序 |

### code_submissions 代码提交记录

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 主键 |
| user_id | UUID | FK → users, NOT NULL | 提交用户 |
| language_id | VARCHAR(32) | FK → languages, NOT NULL | 使用语言 |
| code_hash | VARCHAR(64) | FK → execution_cache, NOT NULL | 代码哈希，关联缓存 |
| status | VARCHAR(16) | NOT NULL | 枚举：pending/running/success/failed/timeout |
| cache_hit | BOOLEAN | NOT NULL, DEFAULT false | 是否命中缓存 |
| cpu_time_ms | BIGINT | | 实际CPU耗时（缓存命中时为NULL） |
| created_at | TIMESTAMPTZ | NOT NULL, DEFAULT NOW() | 提交时间 |

### execution_cache 执行缓存

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| code_hash | VARCHAR(64) | PK | SHA256(language_id + code) |
| language_id | VARCHAR(32) | FK → languages, NOT NULL | 语言 |
| code | TEXT | NOT NULL | 代码内容 |
| stdout | TEXT | | 标准输出 |
| stderr | TEXT | | 标准错误 |
| exit_code | INT | | 退出码 |
| cpu_time_ms | BIGINT | | 执行耗时 |
| created_at | TIMESTAMPTZ | NOT NULL, DEFAULT NOW() | 首次执行时间 |

### templates 模板主表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 主键 |
| slug | VARCHAR(128) | UNIQUE, NOT NULL | URL标识符 |
| title | VARCHAR(255) | NOT NULL | 标题 |
| description | TEXT | NOT NULL | 说明（Markdown） |
| category | VARCHAR(64) | | 分类 |
| like_count | INT | NOT NULL, DEFAULT 0 | 点赞数冗余 |
| created_at | TIMESTAMPTZ | NOT NULL, DEFAULT NOW() | 创建时间 |
| updated_at | TIMESTAMPTZ | NOT NULL, DEFAULT NOW() | 更新时间 |

### template_codes 模板多语言代码

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 主键 |
| template_id | UUID | FK → templates, NOT NULL | 关联模板 |
| language_id | VARCHAR(32) | FK → languages, NOT NULL | 语言 |
| code | TEXT | NOT NULL | 示例代码 |

联合唯一约束：(template_id, language_id)

### template_likes 点赞

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| user_id | UUID | PK, FK → users | 用户 |
| template_id | UUID | PK, FK → templates | 模板 |
| created_at | TIMESTAMPTZ | NOT NULL, DEFAULT NOW() | 点赞时间 |

### template_comments 评论

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 主键 |
| template_id | UUID | FK → templates, NOT NULL | 关联模板 |
| user_id | UUID | FK → users, NOT NULL | 评论者 |
| content | TEXT | NOT NULL | 评论内容 |
| created_at | TIMESTAMPTZ | NOT NULL, DEFAULT NOW() | 评论时间 |
| updated_at | TIMESTAMPTZ | | 编辑时间 |

### user_favorites 收藏

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| user_id | UUID | PK, FK → users | 用户 |
| template_id | UUID | PK, FK → templates | 模板 |
| created_at | TIMESTAMPTZ | NOT NULL, DEFAULT NOW() | 收藏时间 |

### admin_operation_logs 管理员操作日志

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK | 主键 |
| operation_type | VARCHAR(64) | NOT NULL | 操作类型 |
| target_table | VARCHAR(64) | | 目标表 |
| target_id | UUID | | 目标记录ID |
| old_value | JSONB | | 修改前的值 |
| new_value | JSONB | | 修改后的值 |
| note | TEXT | | 操作备注 |
| created_at | TIMESTAMPTZ | NOT NULL, DEFAULT NOW() | 操作时间 |

## 建表语句

```sql
-- 启用扩展
CREATE EXTENSION IF NOT EXISTS pgcrypto;
CREATE EXTENSION IF NOT EXISTS pg_trgm;

-- 用户主表
CREATE TABLE users (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    username VARCHAR(64) NOT NULL,
    email VARCHAR(255) NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    avatar BYTEA,
    avatar_mime_type VARCHAR(32),
    bio TEXT,
    level VARCHAR(16) NOT NULL DEFAULT 'standard',
    status VARCHAR(16) NOT NULL DEFAULT 'active',
    banned_reason TEXT,
    banned_until TIMESTAMPTZ,
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    CONSTRAINT users_username_unique UNIQUE (username),
    CONSTRAINT users_email_unique UNIQUE (email),
    CONSTRAINT users_level_check CHECK (level IN ('standard', 'enhanced')),
    CONSTRAINT users_status_check CHECK (status IN ('active', 'banned', 'suspended'))
);

CREATE INDEX idx_users_username_trgm ON users USING gin (username gin_trgm_ops);

-- 升级申请
CREATE TABLE level_upgrade_requests (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID NOT NULL,
    status VARCHAR(16) NOT NULL DEFAULT 'pending',
    reason TEXT,
    reviewer_note TEXT,
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    reviewed_at TIMESTAMPTZ,
    CONSTRAINT level_upgrade_requests_user_fk FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    CONSTRAINT level_upgrade_requests_status_check CHECK (status IN ('pending', 'approved', 'rejected'))
);

CREATE INDEX idx_level_upgrade_requests_user_id ON level_upgrade_requests(user_id);
CREATE INDEX idx_level_upgrade_requests_status ON level_upgrade_requests(status);

-- 配额追踪
CREATE TABLE user_quota_usage (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID NOT NULL,
    period_start TIMESTAMPTZ NOT NULL,
    cpu_time_used_ms BIGINT NOT NULL DEFAULT 0,
    CONSTRAINT user_quota_usage_user_fk FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    CONSTRAINT user_quota_usage_user_period_unique UNIQUE (user_id, period_start)
);

-- 语言配置
CREATE TABLE languages (
    id VARCHAR(32) PRIMARY KEY,
    display_name VARCHAR(64) NOT NULL,
    version VARCHAR(32),
    is_interpreted BOOLEAN NOT NULL DEFAULT true,
    file_extension VARCHAR(16) NOT NULL,
    enabled BOOLEAN NOT NULL DEFAULT true,
    timeout_ms INT NOT NULL DEFAULT 30000,
    sort_order INT NOT NULL DEFAULT 0
);

-- 执行缓存
CREATE TABLE execution_cache (
    code_hash VARCHAR(64) PRIMARY KEY,
    language_id VARCHAR(32) NOT NULL,
    code TEXT NOT NULL,
    stdout TEXT,
    stderr TEXT,
    exit_code INT,
    cpu_time_ms BIGINT,
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    CONSTRAINT execution_cache_language_fk FOREIGN KEY (language_id) REFERENCES languages(id) ON DELETE RESTRICT
);

-- 代码提交记录
CREATE TABLE code_submissions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID NOT NULL,
    language_id VARCHAR(32) NOT NULL,
    code_hash VARCHAR(64) NOT NULL,
    status VARCHAR(16) NOT NULL,
    cache_hit BOOLEAN NOT NULL DEFAULT false,
    cpu_time_ms BIGINT,
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    CONSTRAINT code_submissions_user_fk FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    CONSTRAINT code_submissions_language_fk FOREIGN KEY (language_id) REFERENCES languages(id) ON DELETE RESTRICT,
    CONSTRAINT code_submissions_cache_fk FOREIGN KEY (code_hash) REFERENCES execution_cache(code_hash) ON DELETE RESTRICT,
    CONSTRAINT code_submissions_status_check CHECK (status IN ('pending', 'running', 'success', 'failed', 'timeout'))
);

CREATE INDEX idx_code_submissions_user_id ON code_submissions(user_id);
CREATE INDEX idx_code_submissions_created_at ON code_submissions(created_at DESC);
CREATE INDEX idx_code_submissions_user_created ON code_submissions(user_id, created_at DESC);

-- 模板主表
CREATE TABLE templates (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    slug VARCHAR(128) NOT NULL,
    title VARCHAR(255) NOT NULL,
    description TEXT NOT NULL,
    category VARCHAR(64),
    like_count INT NOT NULL DEFAULT 0,
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    CONSTRAINT templates_slug_unique UNIQUE (slug)
);

CREATE INDEX idx_templates_category ON templates(category);
CREATE INDEX idx_templates_like_count ON templates(like_count DESC);

-- 模板多语言代码
CREATE TABLE template_codes (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    template_id UUID NOT NULL,
    language_id VARCHAR(32) NOT NULL,
    code TEXT NOT NULL,
    CONSTRAINT template_codes_template_fk FOREIGN KEY (template_id) REFERENCES templates(id) ON DELETE CASCADE,
    CONSTRAINT template_codes_language_fk FOREIGN KEY (language_id) REFERENCES languages(id) ON DELETE RESTRICT,
    CONSTRAINT template_codes_template_language_unique UNIQUE (template_id, language_id)
);

-- 点赞
CREATE TABLE template_likes (
    user_id UUID NOT NULL,
    template_id UUID NOT NULL,
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    PRIMARY KEY (user_id, template_id),
    CONSTRAINT template_likes_user_fk FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    CONSTRAINT template_likes_template_fk FOREIGN KEY (template_id) REFERENCES templates(id) ON DELETE CASCADE
);

-- 评论
CREATE TABLE template_comments (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    template_id UUID NOT NULL,
    user_id UUID NOT NULL,
    content TEXT NOT NULL,
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_at TIMESTAMPTZ,
    CONSTRAINT template_comments_template_fk FOREIGN KEY (template_id) REFERENCES templates(id) ON DELETE CASCADE,
    CONSTRAINT template_comments_user_fk FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE
);

CREATE INDEX idx_template_comments_template_id ON template_comments(template_id);
CREATE INDEX idx_template_comments_created_at ON template_comments(template_id, created_at DESC);

-- 收藏
CREATE TABLE user_favorites (
    user_id UUID NOT NULL,
    template_id UUID NOT NULL,
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    PRIMARY KEY (user_id, template_id),
    CONSTRAINT user_favorites_user_fk FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    CONSTRAINT user_favorites_template_fk FOREIGN KEY (template_id) REFERENCES templates(id) ON DELETE CASCADE
);

-- 管理员操作日志
CREATE TABLE admin_operation_logs (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    operation_type VARCHAR(64) NOT NULL,
    target_table VARCHAR(64),
    target_id UUID,
    old_value JSONB,
    new_value JSONB,
    note TEXT,
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE INDEX idx_admin_operation_logs_type ON admin_operation_logs(operation_type);
CREATE INDEX idx_admin_operation_logs_created_at ON admin_operation_logs(created_at DESC);
CREATE INDEX idx_admin_operation_logs_target ON admin_operation_logs(target_table, target_id);

-- 初始语言数据
INSERT INTO languages (id, display_name, version, is_interpreted, file_extension, enabled, timeout_ms, sort_order) VALUES
    ('python', 'Python', '3.12', true, '.py', true, 30000, 1),
    ('lua', 'Lua', '5.4', true, '.lua', true, 30000, 2),
    ('rust', 'Rust', '1.75', false, '.rs', true, 60000, 3);
```
