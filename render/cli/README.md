# `ds4viz-cli`  

一个命令行工具, 渲染代码描述的数据结构.  

```
ds4viz-cli --ir <TOML IR> [--code <代码文件>] [--play <loop>] [--view <{code|graph|both}>] [--step <step>]
```

## 快捷键  

- `Q`: 退出  
- `A`: 上一步  
- `D`: 下一步  
- `W`: 跳转到第一步  
- `S`: 跳转到最后一步  
- `F`: 跳转到指定步
- `E`: 向左轮转视图  
- `R`: 向右轮转试图  
- `T`: 播放
- `G`: 自动重播
