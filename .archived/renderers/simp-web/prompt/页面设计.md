# ds4viz-simpweb 页面设计  

ds4viz-simpweb是一个简单的Demo页面, 对应ds4viz项目的demo服务器.  

页面设计:  

总架构类似力扣:  

左侧(对应leetcode题目): 可视化结果  
右侧(对应leetcode代码): 输入代码  

可下拉选择语言, 对应每种语言的主题色;  
顶栏包括Logo, 远程/本地; 远程包括服务器地址输入框.  
还有手动上传/下载TOML IR.  
页面各部分正确的使用图标.  

关于信息:  

logo
作者WaterRun  
项目ds4viz  
名称demo, 对应demo服务器
GitHub地址 https://github.com/Water-Run/ds4viz  
GPL协议  

矫正包导入开关(见下)  

默认值:  

语言: Python  
服务器: 远程 http://localhost:9090  
矫正包导入: 开启  

对于每种语言, 需要有基本的语法高亮和补全; 选择时有默认对应的ds4viz模板代码.  
语法高亮和补全包括对ds4viz库的高亮补全.  
当自动包导入开关开启时:  

- 如果没有导包语句->添加导包语句  
- 自动替换js和c的导包(如果是`ds4viz.c`/`ds4viz.h`), 添加前缀`../../library/javascript|c`  
比如, C导包:  

```c
#include "ds4viz.c"
```

自动转化  

```c
#include "../../library/c/ds4viz.c"
```

对应服务器的路径. 不开启时无此行为.
